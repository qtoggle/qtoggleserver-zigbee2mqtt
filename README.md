## About

This is an addon for [qToggleServer](https://github.com/qtoggle/qtoggleserver).

It provides ports for Zigbee devices controlled by [Zigbee2MQTT](https://www.zigbee2mqtt.io/).


## Install

Install using pip:

    pip install qtoggleserver-zigbee2mqtt


## Usage

##### `qtoggleserver.conf:`
``` ini
...
peripherals = [
    ...
    {
    }
    ...
]
...
```
